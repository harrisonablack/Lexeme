# Lexeme
A modern native IOS dictionary focused on clarity and typography for readers!

## Why?

I'm a reader!

Often when i read, i have a dictionary beside me. Whether reading philosophy, psychology or
technical writing, I frequently encounter unfamiliar words and concepts. I've used many dictionary apps in the past, however it feels as if they're centered around engagement, retention and monetisation.

I'm a fan of single purpose apps. When i want to edit code, [nvim](https://neovim.io/), when i want to take notes, [obsidian](https://obsidian.md/). I want a dictionary to define words, nothing more.
